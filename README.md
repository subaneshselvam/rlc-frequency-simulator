# RLC Frequency Response Simulator

This project is an interactive RLC circuit simulator developed using Python and Streamlit.

## Features
- Calculates resonance frequency
- Displays magnitude and phase response
- Computes Q-factor and bandwidth
- Supports Series and Parallel RLC circuits

## Technologies Used
- Python
- Streamlit
- NumPy
- Matplotlib

## How to Run
1. Install dependencies:
   pip install streamlit numpy matplotlib

2. Run the project:
   streamlit run rlc_simulator.py
